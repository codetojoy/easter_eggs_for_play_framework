### Summary

* working demo for [hCaptcha](https://www.hcaptcha.com/)

### usage

* recommend: JDK `11.0.26-amzn` via SDKMan!
	* e.g. `sdk env`
* `. ./test_setvars.sh`
* `./run.sh`
* visit http://localhost:9000

### key files

* front-end: `app/views/login.scala.html`
* back-end: `app/controllers/AuthController.authenticate()`
